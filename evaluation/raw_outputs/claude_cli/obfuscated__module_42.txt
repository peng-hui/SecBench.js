The nativeZip function in `lib/bestzip.js` builds a shell command string by concatenating unescaped `destination` and `source` values directly into a `zip --quiet --recurse-paths <dest> <source>` command, then executes it with `child_process.exec`. Since `source`/`destination` are attacker-influenced (CLI args or API caller-supplied paths) and never sanitized or escaped, shell metacharacters (`;`, `` ` ``, `$()`, `&&`, etc.) in either value let an attacker inject arbitrary shell commands.
